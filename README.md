# 📚 Caderno Temático: Introdução ao Machine Learning com NotebookLM

## 🎯 Contexto e Objetivos

Este projeto tem como objetivo explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa, utilizando o NotebookLM para criar um caderno temático estruturado sobre **Machine Learning**.

📌 **Tema escolhido:** Introdução ao Machine Learning

🎯 **Objetivos de estudo:**

* Compreender os conceitos fundamentais de Machine Learning

* Aplicar pensamento crítico no uso de IA para estudo

* Organizar o conhecimento de forma clara e reutilizável

### 🔄 Variações testadas

* Uso de linguagem simples vs técnica
* Solicitação de exemplos práticos
* Pedidos de explicação passo a passo
* Uso de analogias para facilitar entendimento

---

### ⚠️ Dificuldades encontradas ("cicatrizes")

* Respostas muito técnicas → ajustado pedindo linguagem simples
* Falta de exemplos → resolvido solicitando exemplos explícitos
* Conteúdo genérico → melhorado com prompts mais específicos

💡 **Aprendizado:** quanto mais claro e detalhado o prompt, melhor a qualidade da resposta da IA.

## 📘 Miniguia de Estudo (Entrega Final)

### 🧾 Resumo Estruturado

**📌 O que é Machine Learning:**
Machine Learning é uma área da Inteligência Artificial que permite que sistemas aprendam a partir de dados, sem serem explicitamente programados para cada tarefa.

**📌 Principais tipos:**

* **Aprendizado Supervisionado:** usa dados rotulados (ex: prever preços)
* **Aprendizado Não Supervisionado:** encontra padrões em dados (ex: agrupamentos)
* **Aprendizado por Reforço:** aprende com tentativa e erro (ex: jogos e robótica)

**📌 Como funciona:**

1. Coleta de dados
2. Treinamento do modelo
3. Teste e validação
4. Uso em cenários reais

**📌 Exemplos práticos:**

* Recomendação de filmes (Netflix)
* Filtros de spam em e-mails
* Reconhecimento facial
* Assistentes virtuais

---

### 📖 Glossário

* **Algoritmo:** conjunto de regras para resolver um problema
* **Modelo:** resultado do treinamento de um algoritmo
* **Dataset:** conjunto de dados usado para treinar o modelo
* **Overfitting:** quando o modelo aprende demais os dados e perde generalização
* **Feature:** característica usada como entrada no modelo
